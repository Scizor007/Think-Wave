pH Scale: Basics

Understanding the pH Scale

The pH scale is used to measure how acidic or basic a substance is. It ranges from 0 to 14:

pH Scale Breakdown

  • Acids (pH 0-6): These substances release hydrogen ions (H⁺) in a solution. Example: Lemon juice, vinegar.
  • Neutral (pH 7): Pure water is a neutral substance, meaning it has a balanced number of H⁺ and OH⁻ ions.
  • Bases (pH 8-14): These substances release hydroxide ions (OH⁻) in a solution. Example: Soap, baking soda.
